## 从代码运行

1. `git clone git@github.com:zorroe/HaloMusic.git`
2. `cd HaloMusic`
3. `npm i`
4. `npm run dev`

>electron 国内镜像 `npm config set electron_mirror=https://registry.npmmirror.com/-/binary/electron/`
